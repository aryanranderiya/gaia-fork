# @heygaia/ui

Production-ready UI components for building AI assistants and chatbots — the same components that power [GAIA](https://heygaia.io).

This package lives in its own repository: **[github.com/theexperiencecompany/gaia-ui](https://github.com/theexperiencecompany/gaia-ui)**

## Usage

```bash
npx @heygaia/ui add chat-bubble
npx @heygaia/ui add raised-button tool-calls-section
```

Or via the shadcn CLI:

```bash
npx shadcn@latest add https://ui.heygaia.io/r/chat-bubble.json
```
