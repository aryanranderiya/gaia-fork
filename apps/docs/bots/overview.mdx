## Quick Start

1. Choose your platform
2. Follow the setup guide for that platform
3. Link your GAIA account using the `/auth` command
4. Start using GAIA from your messaging app!

## Common Commands

Across all platforms, these commands work the same way:

| Command | Description |
|---------|-------------|
| `/gaia <message>` | Chat with GAIA |
| `/auth` | Link your account |
| `/workflow list` | List workflows |
| `/todo list` | List todos |
| `/weather <location>` | Get weather |
| `/search <query>` | Search your data |

See the [Commands Reference](/bots/commands) for full details.

## Authentication

Before using most commands, you need to link your account:

1. Use the `/auth` command in your bot
2. Click the authentication link provided
3. Log in to your GAIA account
4. Your messaging account will be linked automatically

This linking is secure and allows GAIA to access your data on your behalf.
