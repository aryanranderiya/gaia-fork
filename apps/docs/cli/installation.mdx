## Install the CLI

The GAIA CLI (`@heygaia/cli`) is a terminal-based tool for setting up, configuring, and managing GAIA instances.

<Tabs>
  <Tab title="curl (Recommended)" icon="terminal">
    The quickest way to install. Uses npm if available, falls back to bun, or installs Node.js automatically.

    ```bash
    curl -fsSL https://heygaia.io/install.sh | sh
    ```
  </Tab>

  <Tab title="npm">
    ```bash
    npm install -g @heygaia/cli
    ```
  </Tab>

  <Tab title="npx (No Install)" icon="play">
    Run directly without installing:

    ```bash
    npx @heygaia/cli init
    ```
  </Tab>

  <Tab title="pnpm">
    ```bash
    pnpm add -g @heygaia/cli
    ```
  </Tab>

  <Tab title="bun">
    ```bash
    bun add -g @heygaia/cli
    ```
  </Tab>
</Tabs>

## Verify Installation

```bash
gaia --version
gaia --help
```

## Requirements

- **macOS or Linux** (Windows via WSL2)
- **Docker** installed and running
- **Git** installed

The CLI will check for these prerequisites and guide you through installing anything missing.
