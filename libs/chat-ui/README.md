# @heygaia/chat-ui

GAIA chat UI components, extracted from `gaia/apps/web` so they can be used across surfaces — the GAIA web app, the motion studio (Remotion), and any future demo or recording flow — without duplication.

## Install

```bash
pnpm add @heygaia/chat-ui
```

## Usage

```tsx
import { ChatBubbleUser, ChatBubbleBot } from "@heygaia/chat-ui";
import "@heygaia/chat-ui/styles.css";
```

## Status

Early — components are being migrated incrementally on the `feat/chat-ui-package` branch.
